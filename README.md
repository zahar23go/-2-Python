
# Python - Урок 4: Работа со строками

Проект содержит утилиты для работы со строками и тесты к ним.

## Файлы
- `string_utils.py` - класс StringUtils с методами для обработки строк
- `test_string_utils.py` - unit-тесты с использованием pytest
- `test_all_methods.py` - демонстрация всех методов
- `test_usage.py` - примеры использования
- `defects.txt` - список дефектов/задач
