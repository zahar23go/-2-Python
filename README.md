# Python - Урок 4: Работа со строками

Проект содержит утилиты для работы со строками и тесты к ним.

## Файлы
- string_utils.py - класс StringUtils с методами для обработки строк
- test_string_utils.py - unit-тесты с использованием pytest
- test_all_methods.py - демонстрация всех методов
- test_usage.py - примеры использования
- defects.txt - список дефектов/задач

## Установка и запуск
1. Клонировать репозиторий: git clone https://github.com/zahar23go/-2-Python.git
2. Перейти в папку: cd -2-Python
3. Переключиться на ветку урока 4: git checkout lesson4
4. Запустить тесты: pytest

## Методы StringUtils
1. capitalize(string) - делает первую букву заглавной
2. trim(string) - удаляет пробелы в начале строки
3. contains(string, symbol) - проверяет наличие символа
4. delete_symbol(string, symbol) - удаляет символ из строки
5. reverse_string(string) - переворачивает строку
